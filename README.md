# TEST_Gjango_API
DJANGO REST Test project 
REST-сервис просмотра текущей зарплаты и даты следующего повышения. Каждый сотрудник может видеть только свою сумму. 
Для обеспечения безопасности, реализован метод где по логину и паролю сотрудника будет выдан секретный токен, 
который действует в течение определенного времени. 
Запрос данных о зарплате должен выдается только при предъявлении валидного токена.

Реализован исключительно на Django
